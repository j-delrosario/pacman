## Mock Airline Ticket Booking Service

### Setup

Change paramters in main to your db and credentials e.g.
`connectDB("jdbc:postgresql://localhost:<port>/<db>", "<user>", "<password>");`

### Run

Run a psql instance

```
> javac Assignment2.java
> java Assignment2
```
